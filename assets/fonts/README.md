# Sofia Pro Font Files

## Required Files

Download the following Sofia Pro font files and place them in this directory:

1. **SofiaProSoftReg.woff2** - Regular weight
2. **SofiaProSoftMed.woff2** - Medium weight  
3. **SofiaProSoftBold.woff2** - Bold weight

## Font Sources

You can download Sofia Pro from:
- **Font Squirrel**: https://www.fontsquirrel.com/fonts/sofia-pro (free for personal use)
- **OnlineWebFonts.com**: Search for "Sofia Pro" and download WOFF2 format
- **Note**: For commercial use, you may need to purchase a license from Mostardesign Studio

## File Format

- **Web**: Use `.woff2` format (already in this directory after download)
- **Native (iOS/Android)**: Font files will be automatically loaded from Expo

## Usage

After downloading the fonts to this directory, run:
```bash
npx expo install expo-font
```

The fonts will be automatically configured in the app.
